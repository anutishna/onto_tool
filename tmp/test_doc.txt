Реализация СЛУ должна предусматривать следующие состояния:
- состояние завершенности процесса;
- инициализация;
- движение вперед;
- аварийное состояние;
- движение вправо;
- движение назад;
- движение влево;

Переход из состояния «состояние завершенности процесса» в состояние «инициализация» должен осуществляться при следующих условиях:
- старт укладки плитки;
После перехода из состояния «состояние завершенности процесса» в состояние «инициализация» должны осуществляться следующие действия:
- установка таймаута;

Переход из состояния «инициализация» в состояние «аварийное состояние» должен осуществляться при следующих условиях:
- превышении времени пребывания в состоянии инициализации;

Переход из состояния «инициализация» в состояние «движение вперед» должен осуществляться при следующих условиях:
- завершение инициализации;
После перехода из состояния «инициализация» в состояние «движение вперед» должны осуществляться следующие действия:
- движение к начальной точке монтажа;

Переход из состояния «движение вперед» в состояние «движение вправо» должен осуществляться при следующих условиях:
- координата y равна номеру витка;
После перехода из состояния «движение вперед» в состояние «движение вправо» должны осуществляться следующие действия:
- увеличение координаты x;
- разворот;
До перехода из состояния «движение вперед» в состояние «движение вправо» должны осуществляться следующие действия:
- укладка плитки;

Переход из состояния «движение вправо» в состояние «движение назад» должен осуществляться при следующих условиях:
- координата x равна номеру витка;
После перехода из состояния «движение вправо» в состояние «движение назад» должны осуществляться следующие действия:
- разворот;
- уменьшение координаты y;
До перехода из состояния «движение вправо» в состояние «движение назад» должны осуществляться следующие действия:
- укладка плитки;

Переход из состояния «движение назад» в состояние «движение влево» должен осуществляться при следующих условиях:
- координата y по модулю на единицу номера витка;
После перехода из состояния «движение назад» в состояние «движение влево» должны осуществляться следующие действия:
- разворот;
- уменьшение координаты y;
До перехода из состояния «движение назад» в состояние «движение влево» должны осуществляться следующие действия:
- укладка плитки;
